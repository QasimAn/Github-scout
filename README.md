# GitHub Scout

GitHub Scout is a lightweight, responsive frontend web application designed to look up any GitHub user and instantly display their profile metrics. Built using vanilla web technologies, this project demonstrates asynchronous data fetching, REST API integration, and clean, state-driven UI updates.

## 🚀 Features
* **Real-Time Data Fetching:** Integrates with the official GitHub REST API using modern asynchronous JavaScript (`async/await`).
* **Dynamic UI Updates:** Manipulates the DOM on-the-fly to render user avatars, bios, follower counts, and public repository statistics without refreshing the page.
* **Error Handling:** Features robust input validation and error catch blocks to handle invalid usernames or API rate limits gracefully.
* **Responsive Design:** Styled with modern CSS (Flexbox/Grid) to ensure a seamless dashboard experience across mobile, tablet, and desktop devices.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **API:** GitHub REST API v3
